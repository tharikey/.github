**ThariKey** is a toolkit for writing Dhivehi. At its core is a fast, **lexicon-free** Thaana
engine — one shared brain that drives native input methods, live transliteration, and a curated
font library, so typing Thaana feels the same everywhere you do it.

### The pieces

- **Engine** — a lexicon-free Thaana keyboard &amp; transliteration core, written in Rust, with
  native and WebAssembly bindings ([`@tharikey/engine`](https://www.npmjs.com/package/@tharikey/engine) on npm).
- **macOS** — a native Thaana input method with a layout customizer and font browser.
- **Windows** — a native Thaana keyboard (TSF). &nbsp;<sub>_coming soon_</sub>
- **Android** — a Thaana soft-keyboard for phones. &nbsp;<sub>_coming soon_</sub>
- **Fonts** — a curated, co-branded library of Thaana typefaces, installable in a tap.
- **Website** — downloads, setup guides, and developer documentation.

Everything is driven by the same TOML-defined schemes and the same engine, so a fix or a new
layout lands everywhere at once.
