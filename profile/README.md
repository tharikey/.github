<h1 align="center">ThariKey</h1>

<p align="center"><em>Bringing sanity to Thaana input across the board.</em></p>

---

ThariKey is a toolkit for writing Dhivehi — a fast, data-driven Thaana **input method**, a curated
**font library**, and the lexicon-free **engine** that powers them.

🌐 **[tharikey.com](https://tharikey.com)** · _launching soon_

### The pieces

- **Engine** — a lexicon-free Thaana keyboard & transliteration core, with native and WebAssembly bindings.
- **macOS** — a Thaana input method and font browser built on the engine.
- **Fonts** — a curated library of Thaana typefaces.
- **Website** — downloads, guides, and developer documentation.

<sub>Built with care in the Maldives.</sub>
